Auto Zone

React website

It's a Car Manufacturer React Project. It's a react project with react database. It's an e-commerce website for Manufacturers. This project is safe from hackers also. It's a website where we display products and a normal user can buy those products. This website also has an admin route where admins do their work and we add a blog page too for the company to add blogs.

What kind of technologies I used :

  1. on the client-side we used react js
  2. on the server-side we used node js
  3. We used Firebase authentication and deployment
  4. It has a blogging system where admin user's can add a blog
  5. For DataBase we used NoSQL ( MongoDB )
  6. For payment we used Stripe
