Gym Trainer Michael

Trainer Michael

React website

It's a React project. It's a project on a gym portfolio website and it was my first react js project. It's a website where gym trainer can add what training they're going to provide and the user can buy that training classes. It's looks like a portfolio of a trainer but it's a useful business website for a trainer.

What kind of technologies I used :

  1. The website is made with React JS
  2. It has firebase authentication system
  3. It is Mobile Friendly
  4. It has a blogging system where admin user's can add a blog
  5. It has training class to make the user's fit

Live Demo : https://gym-trainer-michael.firebaseapp.com/home
