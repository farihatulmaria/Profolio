Doctor Portal

React website

It's an hospital website for Doctors.In this website the users can apply for and appointment just like their would have done in hospital.User's can even choose a day they want that appointment. Their can cancel an appointment from their dashbord. We have a admin route where the admin can add new doctors, see all of the user's activities and so on . This website also has an admin route where admins do their work and we add a blog page too for the company to add blogs.

What kind of technologies I used :

  1. on the client-side we used react js
  2. on the server-side we used node js
  3. We used Firebase authentication and deployment
  4. It has a blogging system where admin user's can add a blog
  5. For DataBase we used NoSQL ( MongoDB )
  6. For payment we used Stripe
